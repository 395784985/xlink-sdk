## 接口设计

### <a name="outlet_staffs_auth">1. 维修点员工登录认证接口</a>

#### **Request**

*URL*

```
POST /warranty/{appid}/outlet_staffs/auth
```

*Header*

```
Content-Type : "application/json"
Access-Token : "APP调用凭证"
```

*Content*

```
{
  "User_name" : "admin",
  "User_pwd" : "123456"
}
```

#### **Response**

*Header*

```
HTTP/1.1 200 OK
```

*Content*

```
无
```
